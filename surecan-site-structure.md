# Improved Site Structure for Surecan Clinic

## Overview
This structure maps out a professional, clinic-focused website with clear flows for consultations and user engagement. Based on best practices for health sites (e.g., HIPAA-friendly, mobile-first).

## Page Hierarchy
- **Homepage** (/)
  - Hero Section: Welcome message + Book Consult CTA
  - About: Brief clinic overview
  - Services: 3 key services with images
  - Testimonials: Patient reviews
  - Book Now Strip: Urgent CTA
  - Footer: Contact, links, social

- **Services** (/services)
  - Detailed service pages (e.g., /services/initial-consultation)
  - Each with description, pricing, booking link

- **About** (/about)
  - Team bios, clinic history, mission

- **Contact** (/contact)
  - Form + map + hours

- **Blog** (/blog) - Optional for health tips

## User Flows
1. **Booking Flow**: Home → Services → Book Consult → Confirmation
2. **Info Flow**: Home → About → Contact
3. **Emergency Flow**: Footer link to emergency contact

## Technical Structure
- **Frontend**: Wix with Velo for custom JS (forms, animations)
- **Backend**: Wix databases for bookings/patients
- **Integrations**: Twilio for SMS, Google Analytics

## GitHub Repo Structure (if using for versioning)
- /docs: Site structure, wireframes
- /assets: Images, icons
- /code: Velo snippets
- /templates: HTML templates
